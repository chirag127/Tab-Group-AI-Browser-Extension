## Quickstart

1.  Clone the repository: `git clone https://github.com/chirag127/CogniFlow-AI-Tab-Organizer-Browser-Extension.git`
2.  Install dependencies: `npm install`
3.  Start the backend server: `npm start --prefix backend`
4.  Load the extension in your browser.
