# Security Policy

The security of the CogniFlow AI Tab Organizer is a top priority. We appreciate the efforts of security researchers and the community to help us maintain a high standard of security.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to us immediately by emailing `security@example.com`. **Please do not disclose the vulnerability publicly until we have had a chance to address it.**

When reporting a vulnerability, please provide the following information:

*   A clear and concise description of the vulnerability.
*   Steps to reproduce the vulnerability.
*   The version of the extension you are using.
*   Any relevant screenshots or code snippets.

We will do our best to respond to your report within 48 hours and provide you with a timeline for addressing the vulnerability. We appreciate your patience and cooperation.

## Supported Versions

We provide security updates for the latest version of the extension. We encourage all users to keep the extension up to date.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Scope

This security policy applies to the `CogniFlow-AI-Tab-Organizer-Browser-Extension` repository.

## Out of Scope

This security policy does not cover the following:

*   Vulnerabilities in third-party dependencies. Please report these to the respective projects.
*   Vulnerabilities that are not reproducible.
*   Vulnerabilities that are already publicly known.
